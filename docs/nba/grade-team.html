Win / Loss Rolling Record

trend_data <- trend_data %>%
  mutate(win_flag = ifelse(result == "Win", 1, 0),
         cumulative_wins   = cumsum(win_flag),
         cumulative_losses = cumsum(1 - win_flag))

trend_long <- trend_data %>%
  select(game_num, cumulative_wins, cumulative_losses) %>%
  pivot_longer(cols = c(cumulative_wins, cumulative_losses),
               names_to = "Type", values_to = "Count") %>%
  mutate(Type = recode(Type, cumulative_wins="Wins", cumulative_losses="Losses"))

ggplot(trend_long, aes(x = game_num, y = Count, color = Type)) +
  geom_line(linewidth = 1.1) +
  scale_color_manual(values = c("Wins"="#2ecc71","Losses"="#e74c3c")) +
  labs(
    title    = paste(team_name_full, "— Cumulative Win/Loss Record"),
    x        = "Game Number",
    y        = "Games",
    color    = NULL
  ) +
  theme_minimal(base_size = 13) +
  theme(legend.position = "top")

How the Grade Is Calculated

Each category is graded against NBA league averages from the 2024-25 season and converted to a 4.0 scale. Scoring and defense each carry 15% of the overall grade. Field goal percentage accounts for 12%, while win percentage, ball security, rebounding, and assists each account for 10%.

The grade scale is: A+ (elite) · A / A- (excellent) · B range (above average) · C range (league average) · D range (below average) · F (well below average).
